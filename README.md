# Location Lookup App

## Overview
This project is a web application that allows users to search by ZIP code and retrieve branch, team, and regional assignment data.

## Tech Stack
- Python (Flask API)
- SQL Database
- REST APIs
- React (planned)

## Features
- ZIP code search
- Branch and team mapping
- API-based lookup
- Scalable backend design

## Purpose
This project is designed to improve operational efficiency by providing quick access to location-based organizational data.

## Status
In progress – building backend API and database integration.
